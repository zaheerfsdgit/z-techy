you can always look back at previous versions or undo mistakes if needed.

Steps to Setup a Git 


-- First Download the Git from official Git Web Site
-- Then Install the Git.

Now, Configure the Git and Set up your username and eamil as below;

>> git config --global user.name "Your Name"
>> git config --global user.email "your.email@example.com"

Now, check version of Git and check Configure details:

>>git --version
==============> to check git version

>>git config --global --list
=============> to check user name and user email details

Create a Repository 


-- Navigate your project directory and initialize a Git Repository.

-- Use below command.

>> git init

Make First Commit 


>> git status

-- git status command Shows the status of changes in your working directory.
-- it is also used to check staging and tracking status.

So that we use "add" command.

-- Adding files to the staging area and commit your changes.
-- Use below command.

>> git add .  OR >> git add file name OR git add *

-- By using add command file get into staging and tracking status.

*** So Now check git status, 
*** Now it seems that "on master branch" but with no commits

-- So, Use no commit command

>> git commit -m "First Commit"

-- Now check the status wiht >> git status

-- Log command used to check all commits list / details 

>> git log OR >> git log --oneline

-- diff command 

>> git diff

-- It shows the changes that we have done in coding in between the commits 
-- by taking commit's hash values.

Push Project into Git Repository Process 


-- Once you done with basic commands like 
-- init, status, add, commit log then now do Push

--- Now go to Git Hub and Create new Repository So that you will be 
get below details, now come to Git Bash and use below commands
to push your project to Git Repository - you should need your Git Hub account..

git branch -M main
git remote add origin https://github.com/zaheerfsdgit/testing-123.git
git push -u origin main

** Now use above commands one by one in Git Bash.
