About Array List


In Java, an ArrayList is a resizable array implementation of the List interface, 
found within the java.util package. It provides a dynamic array that can grow or shrink 
in size as elements are added or removed, unlike traditional arrays which have a fixed size.

Array List Example Program 


class arraylist1
{
 public static void main(String args[])
  {
    ArrayList al1 = new ArrayList<>(List.of(44,55,66));
    System.out.println(al1);

    al1.add(101); // adding a new element
        System.out.println(al1);
    
    al1.add(0,33); // inserting a new element
        System.out.println(al1);

    al1.set(3,77); // modifying an element in list
        System.out.println(al1);

    al1.remove(1) // remving an element by index
        System.out.println(al1);
    
    System.out.println(al1.contains(44));
        // searching a element in List
        // you get true or falise

    System.out.println(al1.indexOf(3));
        // provides elements as per the index position
    
    ArrayList al2 = new ArrayList<>(List.of(1,2,3,4,5));
    
    al1.addAll(al2);
    System.out.println(al1);

    // uisng for loop
    for(int i=0;i<=al2.size()-1;i++)
    {
            System.out.println(al2.get(i));
    }

    // uisng for each loop
    for(int i:al2)
    {
            System.out.println(i);
    }
  }
}

About Linked List


In Java, Linked List is a dynamic linear data structure where elements are not stored in contiguous memory locations. 
Instead, each element, referred to as a "node," contains both the data and a reference (or pointer) to the next node in the sequence.
In the case of a doubly-linked list, each node also contains a reference to the previous node.

Linked List Example Program 


class Linkedlist1
{
 public static void main(String args[])
  {
    LinkedList al2 = new LinkedList<>(List.of(44,55,66));
    System.out.println(al2);

    al2.add(101);
    System.out.println(al2);

    al2.add(1,202);
    System.out.println(al2);

    al2.pollFirst();
    System.out.println(al2);

    al2.pollLast();
    System.out.println(al2);

    al2.pop();
    System.out.println(al2);
  }
}
